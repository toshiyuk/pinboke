利用規約・プライバシーポリシー

最終更新日：2026年3月30日

1. サービスの概要

ピンポケ補正ツール（以下「本サービス」）は、アップロードされた画像にシャープ化処理を施し、補正後の画像をユーザーに返すWebツールです。

2. 画像データの取り扱い

本サービスにアップロードされた画像は、サーバー上に一切保存されません。
画像はリクエストを受け取った瞬間にメモリ上で処理され、補正後の画像をレスポンスとして返した時点で即座に破棄されます。
  • アップロードされた画像はディスクに書き込まれません
  • サービス運営者（管理者）も画像を閲覧・取得することはできません
  • 第三者への提供・共有は行いません
  • 画像をAIの学習データとして使用することはありません

3. 収集する情報

本サービスは、ユーザーの個人情報（氏名・メールアドレス・IPアドレス等）を収集・記録しません。 アクセスログ（Renderプラットフォームが自動的に記録するもの）は、サービスの安定稼働のため一定期間保持される場合があります。

4. 禁止事項

  • 第三者の権利を侵害する画像のアップロード
  • 違法・わわいせつ・公序良俗に反する画像のアップロード
  • サービスへの過度な負荷をかける行為（連続大量リクエスト等）

5. 免責事項

本サービスは現状のまま（as-is）提供されます。補正結果の品質・精度について保証するものではありません。 本サービスの利用によって生じたいかなる損害についても、運営者は責任を負いません。

6. サービスの変更・停止

運営者は予告なくサービスの内容を変更・停止する場合があります。

7. 規約の改定

本規約は必要に応じて改定することがあります。改定後の規約はこのページに掲載した時点で効力を生じます。

8. お問い合わせ

本サービスに関するお問い合わせは GitHub Issues よりお願いします。
